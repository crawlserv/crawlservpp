Detailed Description

Algorithm generate random tweet texts from a text corpus.

This is a semi-serious proof-of-concept class for the crawlserv++ Analyzer module.

The implementation of the algorithm itself is done by the slightly modified rawr class, originally by Kelly Rauchenberger – see the GitHub repository. 👌

Warning
This algorithm may use A LOT of memory when parsing large corpora, adjust your swap size accordingly to prevent the server from being killed by the operating system!
